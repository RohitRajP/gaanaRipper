\n \n • What is an \"IP Lookup\" ?\n \n

\n
\n
\n
\n An IP Lookup is defined as finding the geographical location (or related information like currency, timezone etc.) of an IP address. The lookup can be done either with our API or website. For example, with a plan of 60,000 IP Lookups / month, you can find the location of 60,000 IP addresses in a month. Each conversion of IP address to location is counted as a lookup (repeat queries of the same IP address counts as a separate lookup).\n
\n
\n
\n\n
\n
\n

\n
\n
\n
\n We offer subscription plans with monthly & annual billing options. You can choose a plan based on your expected \n monthly IP address lookup requirement. This IP lookup limit can be changed by upgrading or downgrading your plan. \n You'll receive an API key along with usage instructions when you sign-up for a paid plan.\n
